## The Problem

The US staffing and recruiting industry is massive, fragmented, and ripe for AI disruption -- but the current landscape is failing agencies on every front.

| Metric | Value |
|--------|-------|
| US Staffing Market Size | **$189B** |
| Number of Agencies | **25,000+** |
| Agencies Using Some AI | **75%** |
| Agencies That Haven't Seen ROI from AI | **32%** |
| Recruiter Burnout Rate | **81%** |
| First-Year Recruiter Turnover | **90%** |
| Enterprise AI Tool Cost | **$50K - $500K/year** |

**The core issues:**

- Enterprise AI platforms (Eightfold, Beamery, hireEZ) cost $50K-$500K/year, putting them out of reach for the vast majority of SMB agencies
- Off-the-shelf AI tools don't understand staffing workflows and require months of integration
- Recruiters are drowning in administrative work, spending 60%+ of their time on non-revenue tasks
- No one places dedicated AI integration specialists inside agencies to bridge the gap between AI capability and operational reality

---

## The Solution: AI Integrators for Staffing

UAIS recruits, certifies, and places **AI Integrators** directly into staffing agencies. These specialists:

1. **Learn** the agency's existing workflows, tech stack, and pain points
2. **Build** custom AI automation using Claude Code (primary) + n8n (orchestration) + direct ATS APIs
3. **Deploy** 10 purpose-built AI agents tailored to staffing operations
4. **Deliver** measurable productivity gains -- doubling output within the deployment period (30 days for Starter/Professional, 90 days for Enterprise)

> **The model is simple:** The human AI Integrator handles strategy and relationships. The AI agents handle execution at scale. 24/7 autonomous operation. 7 days to deploy. Zero risk.

## Tech Stack

| Layer | Tool | Role |
|-------|------|------|
| **Primary** | [Claude Code](https://www.anthropic.com) | The AI Integrator's main tool. Builds custom agents, direct API integrations, complex reasoning tasks. This is what makes UAIS different. |
| **Secondary** | [n8n](https://n8n.io) (self-hosted) | Orchestration and scheduling layer. Cron jobs, webhook receivers, visual monitoring. The agency-facing dashboard. |
| **Autonomous** | [OpenClaw](https://openclaw.com) | The Operations Hub runtime. Runs 24/7 on dedicated cloud infrastructure. |
| **Connectors** | Direct ATS APIs | Bullhorn, Lever, Greenhouse, JobAdder connected via Claude Code, not middleware. |
| **Scraping** | [Apify](https://apify.com) | Candidate sourcing, market intelligence, company research at scale. |
| **Outbound** | [Instantly](https://instantly.ai) / [Smartlead](https://smartlead.ai) | High-deliverability cold email for client acquisition. |
